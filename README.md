# 画像も加えられる共有スケジュール管理アプリ
**カード型で予定を管理できるWebアプリ**

[実際のウェブアプリ](https://schedule-ecxj.onrender.com)

## 制作背景
Pythonを学ぶ過程で実際に動くものを作ると技術が身につくと思い、
Flaskを使ってスケジュール管理アプリを作りました

## 技術スタック
- Python / Flask
- SQLiete3
- HTML / CSS / Jinja2

## 機能
- 予定を追加・削除・編集ができる
- 予定に画像を追加できる
- カード型で表示
- 編集時にページ遷移してるように見えない設計

## 工夫したこと
- 再読み込みをその場で編集してるように見える
- シンプルに操作ができる設計
- 予定に画像を追加でき、一目でその予定を理解できる
- 画像ファイルはフォルダで管理して、画像のファイル名をDBで管理することで、DBを軽量化をした

## 起動方法
pip install flask tzdata

python app.py
